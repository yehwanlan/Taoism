# 恶报品第三

## 原文

恶报品第三

尔时，太上道君告普济及在会众真人曰：自今后，若有众生毁坏天尊大道形像者，死入九幽十八地狱，万劫受苦；苦尽方生，得生人中，受肿癞病，遍身脓血，臭秽腥膻，人不喜见。訾毁三洞大法经典者，死历九幽，并入酆都；经千百万劫，方遇圣人救拔得出，生野兽中；受无量苦，得生人中，舌根烂坏，常有缠喉之疾，眼目有患。诽谤出家法身者，见世得虫癞病，过去生六畜身。破坏灵观、玄坛、圣像者，见世眉发堕落，身体不全，拖腰摩胯；死入寒冰地狱，受苦满足，堕毒蛇身。不敬天尊、大道、法身者，见世受愚痴、憨矮、癫狂病报，过去生六畜中。不信经法、宿命报对者，见世盲聋，过去生鸟兽中，经无量劫，方可得脱。不信罪福因缘者，见世为奴婢，过去生蛮夷中，难遇正法。轻慢出家法身者，见世跛躄，过去生猪狗中，屎尿之中，常为饮食。轻慢三宝，获罪如是。

侵常住福地者，见世形容臭膻，过去生粪秽中；历千万劫，复生污水中，为细线虫。淫犯出家法身者，见世孤寡，为人憎嫌；过去生厕溷中，为蛆虫。邪淫好色者，见世患风邪、癫狂恶病，过去生豭猪中；经之数劫，生诸般野牛之中。偷盗三宝财物者，见世贫穷；过去生奴婢，复生六畜种种禽兽之中，难复人身。偷盗四辈信施财物者，见世得癫狂病，不闻三宝之名；过去生愚痴中。啖食三宝果实、蔬菜者，见世得翻胃，不能进食；过去生饿虎、毒蛇中，被人打猎，常遭刑宪；复生人中，更逢渴病。偷盗斋食及供养宝器、法食者，见世逢霹雳雷电，恶鬼所伤；过去生蟒蛇身，死堕饿鬼中。偷盗供眷、醮坛器具等物者，见世生下贱，乞索觅食，无饱之身。毁坏大道形像者，见世生饿狗毒虫身，饥寒无告，永不得食；过去生猿猴等身。偷盗经像财物者，见世得癞病，死堕诸地狱，受诸苦恼，万劫千生，不得原赦；过去生两头毒蛇、恶虫身。骂詈出家法师身者，见世得风痫重病，过去生野犴身。凌辱出家法身者，见世遭枷锁、牢狱拷掠，过去生蝇蛆中。破斋犯戒者，见世得咽喉疾闭之病，过去生虎豹饿狼之中。借常住三宝财物不还者，十年十倍，死入酆都地狱，鬼拷罪魂，万劫不得原赦；过去生蛇狮、虮蟟、壁蛛等蛆中。杀猎众生者，见世得短命，长受风病，常遭牢狱苦厄；过去生六畜恶兽中。爱食野兽等肉者，见世值恶病，过去生麋鹿中。嗜酒好色者，见世风狂，逢猛火烧燎；过去生淤泥秽土中。好食荤辛臭秽者，见世得腥膻臭臊身，过去生粪秽中。

持心不坚固、好弄出家人者，见世得猿猴报；死入镬汤地狱，经无量劫后，生诸鼠趣中；复生人道，得手脚挛跛之报，皆因不敬三宝中来。手脚断坏拳缩、一足短者，从不敬三宝中来。两眼双盲者，从不信经法、轻忽三宝中来。舌根烂坏者，从訾毁三洞大乘经教中来。瘖痖不能言语者，从道说出家法师中来。一生以来，多害疮癣，脑疽背痈，徧身癞疮，皮毛孔窍脓血腥膻，癞虫咬啮，昼夜恓惶，求生不得，求死不得者，从诳惑三宝中来。鼻塞不闻香臭者，从好啖酒肉、五辛荤秽、冲突三宝中来。身体烂坏，节节自落者，从诽谤三宝中来。唇𠐻齿落，鼻梁崩裂者，从毁道场中来。曲腰歧胸，偻背短脚者，从唝嗃三宝中来。欺忽三宝，偏眼斜视三宝者，见世受癫狂风邪迷惑病，过去生鳝鱼水蛭中来。见世愚痴者，从不信因缘罪福中来。形容疮肿、丑陋者，从骂詈三宝中来。眼赤口斜，鼻倾不正者，从诽笑出家法身中来。愚痴长病，意志不足者，从诳惑三宝中来。劫贼相逢，杀伤财物者，从劫夺三宝财物中来。水溺漂沉者，从饮酒食腥膻之食，秽慢三宝中来。火烧灼烂者，从损废三宝中来。毒蛇吞噬者，从伤杀出家法身中来。毒蛇恶虫入口鼻者，从恶口咒诅中来。毒蛇恶虫入孔窍中者，从耽淫中来。遇虎狼猛兽噉食者，从阴谋恶毒中来。饥寒冻饿，衣食不充足者，从修斋醮供养三宝、果食道场未散，偷取中来。偷食上真献御之食者，永堕恶趣，常作饿鬼，海边阿修罗鬼报。偷盗三宝常住财物与六亲享受者，今世父母离隔，男女常不同居之报。嫉妒奸淫，破人和合，离人恩爱者，见世受分散之报。贫穷困苦者，从破斋犯戒中来。常为奴婢下贱身者，从偷盗悭贪中来。生世得人憎嫌者，从㤭慢唝嗃中来。生世躭淫，不避亲疏好丑者，从猪狗六畜中来。生世嗜酒猖狂者，从鱼鳖虾蟆水族中来。生世嗜肉好杀者，从毒虫猛兽中来。生世不净臭秽者，从科斗虫中来。身体臭秽不净洁者，从好食五辛猪狗中来。六畜生身者，从杀生抵债中来。野兽獐鹿身者，从食肉偷盗中来。毒蛇蟒蛇身者，从嫉妒毒害中来。饿虎生身者，从欺凌劫夺十来。生厕溷中虫者，从淫欲不净中来。生虾蟆、科斗身者，从不供养三宝中来。生鱼鳖水旅身者，从轻泄大道经宝中来。生小虫蚁身者，从违犯经教中来。猘犬、野猪、豺狼者，从背师弃本中来。生乳𬸧、山雀身者，从评论经教、多口合舌中来。生鸠鸽、老鸦诸鸟身者，从好是作法异常中来。生鹰、鹯鸱鹞鸟等身者，从破戒违经中来。生鳏寡孤独、贫寒困苦者，从淫犯出家法身及堕胎害子中来。形容丑陋者，从诽谤三宝，傲诞出家法身中来。风邪、癞病、狂言者，从布施不还及堕子落胎中来。腹大项细者，从灵观玄坛淫欲中来。牢狱枷锁切身者，从謟毒恶党中来。鞭棒楚挞者，从毒害众生不慈悲中来。猫儿、水獭、臊猫等类生身者，从食肉入灵观玄坛中来。𮔅蝂生身者，从食五种荤辛、侵近三宝中来。豺狼生身者，从贪吝不布施中来。狐貉生身者，从虚诳矫诈中来。鼠狖生身者，从食斋残供、饮酒食肉中来。蜈蚣、蚰蜒生身者，从阴毒妒害中来。蜂虿虺蝎生身者，从咒诅出家法身中来。鼹蜓、𧋍蜴生身者，从欺忽三宝中来。蜘蛛、蚯蚓生身者，从悭贪盗取中来。蚊虻、蚤虱生身者，从慢法轻道中来。边夷下贱、髡头断发、眼深鼻长、身体臭者，从杀生邪淫、不信因缘中来。

如此皆生世所犯罪目，死入地狱，受诸罪讫，更生此身。亦有见世报而生此身者，悉由先世所犯此身也。

普济，汝当心精信奉，明示将来，令诸男女知其恶业。故说颂曰：

悭贪不肯布施者，后世贫穷少衣食。

不敬尊师行轻慢，后生下贱奴婢身。

借贷扺债欺负者，后生牛羊猪狗身。

破斋饮酒食肉者，后生饥寒作饿身。

破戒及怀诸嫉妒，后生鸟兽虫蛇身。

毁坏经像及轻慢，后生虫癞脓烂形。

秽污经像及玄坛，后生丑陋不具足。

诽谤三宝并訾毁，后生无告猪羊狗。

偷盗经像诸财物，后生边夷得下贱。

杀害众生无慈念，后生鱼鳖与六畜。

嫉妒胜已杀贤良，后生恶兽毒蛇身。

恶口咒诅无期度，后生猛虎风狂病。

躭淫好色通骨肉，后生豭猪厕溷蛆。

饮酒食肉及五辛，后生腥臭狂邪病。

毁破天尊诸形像，死入九幽生病癞。

訾毁三洞大经法，死入地狱生野兽。

讪谤出家法身者，见世病癞生六畜。

破坏灵观玄像者，眉发堕落身烂坏。

不敬三宝真圣者，见世愚痴作畜生。

不敬经法与宿命，见世盲聋生鸟兽。

不信罪福因缘者，见世奴婢生六夷。

不信出家生轻忽，见世跛躄为猪狗。

秽污三宝清净地，见世腥臭生粪土。

淫犯出家清净地，见世孤寡生厕溷。

邪淫好色无亲疏，见世风邪生豭猪。

劫夺三宝诸财物，见世贫穷作奴婢。

偷盗常住及侵损，见世风病作牛羊。

费用偷盗信施物，见世癫狂为愚痴。

噉食三宝果核等，见世消渴得瘦病。

偷盗斋食清净供，见世躄𬦣生饿鬼。

偷取斋物将己用，见世饥寒不得食。

凌忽骂詈出家人，见世风痫遭牢狱。

破斋犯戒违禁忌，见世噎病生饿鬼。

损害众生无畏惧，见世短命枷锁厄。

食肉腥臭常血秽，见世百病不离身。

嗜酒猖狂无尊卑，见世猛火恶鬼杀。

好食五辛及臭物，见世腥膻生粪土。

绮语戏弄出家人，见世猿猴生鸟兽。

手足挛缩不伸展，不敬三宝得此报。

手足断坏无支节，不敬尊师得此报。

两目俱盲无所见，不信经法得此报。

舌根坏烂及喑痖，诽谤大乘得此报。

言词謇吃不能语，道说出家得此报。

鼻拥不闻香与臭，好食五辛得此报。

身体烂坏脓血流，毁谤三宝得此报。

唇𠐻齿露鼻梁倒，破坏道场得此报。

举身虫出肉臭烂，算弄三宝得此报。

腰脊曲跌背脚折，欺忽三宝得此报。

癫狂风邪迷惑病，不信因缘得此报。

形容丑陋并疮𤺄，咒诅三宝得此报。

眼赤口偏鼻不正，调戏三宝得此报。

愚痴憨矮无意志，诳惑三宝得此报。

劫贼杀害取财物，侵夺三宝得此报。

大水漂溺沉没者，躭色嗜酒得此报。

猛火烧灼形燋烂，损费三宝得此报。

毒蛇吞噬及𧋍螫，打骂出家得此报。

毒蛇入口及鼻孔，恶口咒诅得此报。

毒蛇恶虫入窍穴，邪淫放荡得此报。

虎狼猛兽噉食者，阴谋恶毒得此报。

饥冻寒饿常不足，侵损常住得此报。

身常虚乏无气力，不持斋戒得此报。

父母男女常离隔，破人和合得此报。

贫穷困苦常饥寒，破坏斋戒得此报。

为人奴婢常辛苦，偷盗悭贪得此报。

为世憎嫌人不喜，㤭慢傲诞得此报。

受身猪狗常噉粪，淫犯六亲得此报。

受身鱼鳖虾蟆身，嗜酒猖狂得此报。

受生毒虫猛兽身，嗜肉杀生得此报。

受生科斗蝼蚁身，不净淫秽得此报。

受生獐鹿野兽身，偷盗杀生得此报。

受生蟒蛇毒龙身，嫉妒毒害得此报。

受生饿虎鹰鹞身，欺负抵债得此报。

受生狂犬狐兔身，背师弃本得此报。

受生鸠鸽𬸧雀身，作法异常得此报。

受生雕鹯鸷鸟身，不用经教得此报。

受生孤寡无男女，堕胎害子得此报。

受生丑陋人憎恶，笑弄出家得此报。

受生癫痫风狂病，布施不还得此报。

受生诸雀鸱鷮身，评论毁教得此报。

受生枷锁常切身，謟毒忠良得此报。

受生棒杖不离身，非法拷楚得此报。

受生猫子貍豹身，食肉入观得此报。

受生𮔅蝂臭秽身，荤秽经像得此报。

受生豺狼狐貉身，悋惜虚矫得此报。

受生狌狖貍鼠身，斋竟食肉得此报。

受生蚰蜒蜣螂身，观中淫欲得此报。

受生蜂虿虺蝎身，咒诅恶口得此报。

受生鼹蜓𧋍蜴身，唝嗃傲诞得此报。

受生蜘蛛蚯蚓身，贪淫自盗得此报。

受生蚊虻蚤虱身，轻道慢法得此报。

## 翻譯

### 現代中文翻譯
**惡報品第三**

那時，太上道君告訴普濟真人和在場的眾位真人說：「從今以後，若有眾生毀壞天尊或大道的形像，死後將墮入九幽十八地獄，承受萬劫的苦難；苦報受盡後才能轉生，即使生在人間，也會得浮腫和麻風病，全身流淌膿血，散發腥臭污穢，人們不喜歡看見。若有詆毀三洞大法經典的人，死後將經歷九幽地獄，並進入酆都鬼城；經過千百萬劫，才能遇到聖人救拔得以出離，但會轉生在野獸之中；承受無量的苦難後，才能再生為人，但舌根會腐爛損壞，時常患有喉嚨的疾病，眼目也會有病患。若有誹謗出家修行人，現世就會得蟲癞病，死後會轉生為六畜之身。若有破壞靈觀、玄壇、聖像的人，現世眉毛和頭髮會脫落，身體不健全，彎腰駝背；死後墮入寒冰地獄，受苦期滿後，會轉生為毒蛇之身。若有不敬天尊、大道、修行人的人，現世會遭受愚痴、身材矮小、癲狂等病報，死後會轉生在六畜之中。若有不信經法、不信宿命報應的人，現世會眼盲耳聾，死後會轉生在鳥獸之中，經過無量劫，才能得以解脫。若有不信罪福因緣的人，現世會成為奴婢，死後會轉生在邊疆蠻夷之地，難以遇到正法。若有輕視怠慢出家修行人，現世會手腳跛瘸，死後會轉生在豬狗之中，常以屎尿為飲食。輕慢三寶，就會獲得這樣的罪報。

若有侵占三寶常住福地的人，現世形容腥臭，死後會轉生在糞穢之中；經歷千萬劫後，又會轉生在污水中，成為細小的線蟲。若有淫亂侵犯出家修行人，現世會孤獨寡居，被人憎恨嫌棄；死後會轉生在廁所之中，成為蛆蟲。若有邪淫好色的人，現世會患上風邪、癲狂等惡疾，死後會轉生在公豬之中；經過數劫後，又會轉生在各類野牛之中。若有偷盜三寶財物的人，現世會貧窮；死後會轉生為奴婢，又會轉生為六畜及種種禽獸，難以再恢復人身。若有偷盜十方信眾布施財物的人，現世會得癲狂病，聽不到三寶的名號；死後會轉生在愚痴之中。若有偷吃三寶的果實、蔬菜的人，現世會得翻胃病，不能進食；死後會轉生在飢餓的老虎、毒蛇之中，被人打獵，時常遭受刑法；再轉生為人時，又會遭遇口渴之病。若有偷盜齋食及供養的寶貴器物、法食的人，現世會遭遇雷擊，被惡鬼所傷；死後會轉生為蟒蛇之身，死後再墮入餓鬼之中。若有偷盜供養眷屬、齋醮壇場的器具等物品的人，現世會出身下賤，靠乞討覓食，永遠吃不飽。若有毀壞大道形像的人，現世會轉生為飢餓的狗或毒蟲之身，飢寒交迫，無處申訴，永遠得不到食物；死後會轉生為猿猴等身。若有偷盜經書、佛像財物的人，現世會得麻風病，死後墮入各地獄，承受各種苦惱，萬劫千生，不得赦免；死後會轉生為兩頭的毒蛇、惡蟲之身。若有辱罵出家法師的人，現世會得風癇等重病，死後會轉生為野外的豺狼之身。若有凌辱出家修行人，現世會遭受枷鎖、牢獄、拷打之苦，死後會轉生在蒼蠅、蛆蟲之中。若有破齋犯戒的人，現世會得咽喉閉塞的疾病，死後會轉生在虎、豹、餓狼之中。若有借貸三寶常住財物不歸還的人，每過十年就要加十倍償還，死後墮入酆都地獄，被鬼卒拷問罪魂，萬劫不得赦免；死後會轉生在蛇、獅、蟣、蟟、壁蛛等蟲豸之中。若有殺生打獵的人，現世會得短命之報，長期遭受風濕病痛，時常遭遇牢獄之苦；死後會轉生在六畜、惡獸之中。若有喜愛吃野獸等肉的人，現世會遭遇惡疾，死後會轉生在麋鹿之中。若有嗜酒好色的人，現世會風流癲狂，遭遇猛火燒身；死後會轉生在淤泥穢土之中。若有喜好吃葷腥臭穢之物的人，現世會身體散發腥膻臭氣，死後會轉生在糞穢之中。

若有心志不堅固、喜歡戲弄出家人的人，現世會得猿猴的果報；死後墮入沸水地獄，經過無量劫後，轉生在各種鼠類之中；再轉生為人道時，會得手腳攣縮跛足的果報。這些都是因為不敬三寶而來。手腳斷裂損壞、拳曲不伸、一腳短小的人，是從不敬三寶中來的。雙眼都盲目的人，是從不信經法、輕忽三寶中來的。舌根腐爛損壞的人，是從詆毀三洞大乘經教中來的。喑啞不能說話的人，是從議論出家法師的是非中來的。一生以來，多患瘡癬、腦疽、背癰，全身長滿癞瘡，皮毛孔竅流淌膿血，散發腥膻之氣，被癞蟲咬噬，晝夜驚恐不安，求生不得，求死不得的人，是從欺騙迷惑三寶中來的。鼻子堵塞，聞不到香臭的人，是從喜好吃酒肉、五辛葷穢之物，並以此觸犯三寶中來的。身體腐爛損壞，關節一節節自行脫落的人，是從誹謗三寶中來的。嘴唇破裂、牙齒脫落，鼻梁崩塌斷裂的人，是從毀壞道場中來的。彎腰、雞胸、駝背、短腳的人，是從喧嘩吵鬧、不敬三寶中來的。欺騙輕忽三寶，用斜眼看待三寶的人，現世會遭受癲狂、風邪、迷惑等病，死後會從鱔魚、水蛭中轉生而來。現世愚痴的人，是從不信因緣罪福中來的。形容相貌生有瘡腫、醜陋不堪的人，是從辱罵三寶中來的。眼睛發紅、口角歪斜，鼻子傾斜不正的人，是從誹謗嘲笑出家修行人中來的。愚痴且長期患病，意志薄弱的人，是從欺騙迷惑三寶中來的。遭遇強盜，被殺害並搶奪財物的人，是從搶劫奪取三寶財物中來的。被水淹沒沉溺的人，是從飲酒、吃腥膻之食，並以此污穢怠慢三寶中來的。被火燒傷灼爛的人，是從損壞廢棄三寶中來的。被毒蛇吞噬的人，是從傷害殺戮出家修行人中來的。毒蛇惡蟲進入口鼻的人，是從惡毒的口舌和詛咒中來的。毒蛇惡蟲進入身體孔竅的人，是從沉溺淫欲中來的。遇到虎狼猛獸被噉食的人，是從陰謀惡毒中來的。飢寒交迫，衣食不充足的人，是從在修持齋醮、供養三寶的道場還未結束時，就偷取果食中來的。偷吃供奉上界真仙的食物的人，將永遠墮入惡道，時常作餓鬼，受海邊阿修羅鬼的果報。偷盜三寶常住財物，與六親眷屬一同享用的人，今世會遭受父母分離、男女不能同居的果報。嫉妒、奸淫，破壞別人和睦，離間別人恩愛的人，現世會遭受妻離子散的果報。貧窮困苦的人，是從破齋犯戒中來的。時常身為奴婢、地位下賤的人，是從偷盜和慳吝貪婪中來的。生生世世都受人憎恨嫌棄的人，是從驕傲自大、喧嘩吵鬧中來的。生生世世都沉溺淫欲，不避親疏好醜的人，是從豬狗等六畜中轉生而來。生生世世都嗜酒猖狂的人，是從魚鱉蝦蟆等水族中轉生而來。生生世世都嗜食肉類、喜好殺生的人，是從毒蟲猛獸中轉生而來。生生世世都不潔淨、散發臭穢的人，是從蝌蚪等蟲類中轉生而來。身體臭穢不潔淨的人，是從喜好吃五辛和豬狗肉中來的。轉生為六畜之身的人，是從殺生以抵償債務中來的。轉生為野獸、獐鹿之身的人，是從吃肉和偷盜中來的。轉生為毒蛇、蟒蛇之身的人，是從嫉妒和毒害中來的。轉生為餓虎之身的人，是從欺凌、搶劫、抵賴中來的。轉生在廁所中的蟲子，是從淫欲不淨中來的。轉生為蝦蟆、蝌蚪之身的人，是從不供養三寶中來的。轉生為魚鱉等水族之身的人，是從輕視洩漏大道經寶中來的。轉生為小蟲、螞蟻之身的人，是從違犯經書教義中來的。轉生為瘋狗、野豬、豺狼的人，是從背叛師長、拋棄根本中來的。轉生為乳燕、山雀之身的人，是從評論經教、多嘴多舌中來的。轉生為鳩、鴿、老鴉等鳥類之身的人，是從喜好標新立異、行為異常中來的。轉生為鷹、鸇、鴟、鷂等猛禽之身的人，是從破戒違經中來的。轉生為鰥寡孤獨、貧寒困苦之人，是從淫犯出家修行人以及墮胎害子中來的。形容相貌醜陋的人，是從誹謗三寶，傲慢對待出家修行人中來的。患有風邪、癞病、言語狂亂的人，是從布施後反悔不給以及墮胎中來的。肚子大脖子細的人，是從在靈觀、玄壇行淫欲中來的。牢獄枷鎖常在身的人，是從諂媚、惡毒、結黨營私中來的。常受鞭打棒擊的人，是從毒害眾生、沒有慈悲心中來的。轉生為貓兒、水獺、臊貓等類，是從吃肉後進入靈觀、玄壇中來的。轉生為蟑螂之身的人，是從吃五種葷辛之物，並以此侵犯靠近三寶中來的。轉生為豺狼之身的人，是從貪婪吝嗇、不肯布施中來的。轉生為狐貉之身的人，是從虛偽欺騙、狡猾詭詐中來的。轉生為鼠類之身的人，是從吃了齋戒後的殘餘供品，又飲酒食肉中來的。轉生為蜈蚣、蚰蜒之身的人，是從陰險毒辣、嫉妒傷害中來的。轉生為蜂、蠍、虺、蝎之身的人，是從用惡言詛咒出家修行人中來的。轉生為鼹鼠、壁虎之身的人，是從欺騙輕忽三寶中來的。轉生為蜘蛛、蚯蚓之身的人，是從慳吝貪婪、偷盜竊取中來的。轉生為蚊、虻、蚤、虱之身的人，是從輕視道法、怠慢大道中來的。轉生為邊疆蠻夷、地位下賤，剃頭斷髮，眼窩深陷，鼻子高長，身體腥臭的人，是從殺生、邪淫、不信因緣中來的。

- **九幽、酆都**: 皆指道教觀念中的地獄或冥府，是罪人受苦之處。

- **法身**: 在此處多指「出家法身」，即出家修行的道士或僧人，他們是三寶的代表，侵犯他們罪過極重。
