# 诵念品第八

## 原文

诵念品第八

大慈大悲普济真人，以业报因缘请问太上道君。尔时，道君告普济曰：昔胜喜天王召道士女官三万六千人，妙化天王召道士女官三千二百人，妙德天王召道士女官一千五百人，清净天王召道士女官一万八千人，灵和天王召道士女官五百六十人，浮和天王召道士女官七万二千人。此六天王各同时于其正殿，建立道场，悬诸宝盖，烧百和宝香，装百宝座，设百味饮食，散万种名华，陈百千妙乐，施诸财宝、金银珠玉、上妙衣服、锦绮绫罗、诸般布帛；一百日中诵念此经，依时行道，恭敬礼拜，持斋燃灯，昼夜不懈。王与皇后并诸王子、宰辅大臣，日夜烧香，散华行道，布施忏悔。国中男女臣人百官，闻王建此福田，各自发心，舍其居宅、所有财物，市诸香油、华果、名衣上服、种种法具，来诣国门，以用布施，供养道士；愿王国土安宁，人民快乐，先亡者离苦生天，逍遥自在。填满道路，王不禁制，任其本心。各各国中生获法利，年命长远，衣食无为，金芝、玉英充满园囿，凤凰、孔雀游戏山林，男女欣欣，惟有诵经。时诸国王并寿千岁，方升上界；大臣宰辅各八百岁，相次而迁。七百劫来，国人犹习道也。

道君曰：若国主有灾，诵念是经；水火刀兵，诵念是经；星宿失度，诵念是经；日月薄蚀，诵念是经；阴阳不调，诵念是经；疫毒流行，诵念是经；疾病死亡，诵念是经。邪精魍魉，魅惚人民，诵念是经；毒螫猛兽，伤害兆民，诵念是经。或在牢狱之中，诵念是经；或得杽械枷锁，诵念是经。一切厄难，诵念是经。若人命过，诵念是经，开度亡人，上生华胥。常诵是经，九祖超升。凡诵是经，功德无量，各随其力，罪业消除，生死蒙恩，兆民欢喜，天下太平。

道君曰：昔有一人受持此经，昼夜诵念，不倦于心。时行山中，遇诸猛虎，围绕左右，竞来抟噬。默诵是经，虎即迸散，无有中伤。复有一人，游行海中，风波振荡，身将涌没。诵念是经，得远彼岸，悉复安全。复有一人，身犯王法，枷锁在身。诵念是经，牢狱解散，不被鞭笞。此经功德，有大威力，不可思议。汝等男女，各宜至诚信心敬奉，克获福利，享祚无穷。

# 持齋品第七

## 繁體中文翻譯

### 道君開示齋戒的重要性

道君說：眾生從無始以來受身輪迴，因為慳貪和嗜慾，飲食酒肉，貪婪無厭；這樣滋養肌膚，培育膿血，導致疾病煩惱，更增加罪業根源。生老病死，暑濕風寒，痛癢酸疼，惡風毒氣進入身體污染形體，百病侵襲，念念無常，晝夜不停。

受生稟賦的識性，各各不同，日夜循環，經歷萬萬次生死：有時處在胞胎中，吸收污穢成為膿血，隨即化消；有時居在母腹中，不能出生，便又破壞；雖然暫時分娩，母子暫時分離，但立即死亡，數量無法計算；有的活一歲兩歲乃至百歲，念念衰敗，衰耗怎能停止，頭痛發熱，萬種煩惱侵擾。

一生之中沒有絲毫安樂，實在是因為廣泛食用污穢濁物，積滿身中，酒肉腐爛肌肉，蒸騰五臟六腑，惡風毒氣吹動形骸，即使金玉也會化消，身形豈能長久。我不飲不食，抱道自然，變化無方，長生不死。常常說經教化，開導度化眾生。凡夫愚迷，不能信受奉行，我常常憂慮思念，憐憫痛心在此，所以設立齋法，令其消滅病苦。

我從前在元始天尊御前，受得三十六部經文，齋法精深微妙，濟度群品，施行在人間，具有完備的成法。凡夫無知，認為是虛假荒誕。如同三清眾聖，久已證得道真，恆常都奉行齋戒，勤修不懈怠，何況你們呢。

如果上等根器的人勤修，長期齋戒清淨居室，誦經燒香，中間不懈怠。無論存亡，隨其愚智程度，制定法則不同，善巧方便，開導度化眾生。六齋十直，八節三元，甲子本命，三會庚申，諸種救護法門，微妙難以思議，都是元始天尊引接下界凡人的方法。我常常勸導教化，普遍使人修行。

### 普濟菩薩請問齋戒功德

普濟問道：不知建立齋法，功德輕重如何？又有什麼神明監臨？是單純斷食，還是另有威儀？又至道圓通，遍於一切有法，應當隨分力量，任意能行，設立六齋等法。我所不了解的，願您垂示教誨。

### 道君詳述齋法分類和功德

道君回答說：功德輕重，諸天降臨；威儀軌則，六齋等差別，都備載在靈寶諸經中，現在粗略說明，你應該仔細聽。

齋戒有兩種不同：一是極道，二是濟度。

**極道齋：** 為了發心學道，從初到終，念念持齋，心心不退。又有二門，一叫忘心，二叫滅心。忘心者，始終運用心意，行坐忘形，寂靜如死灰，同於枯木，滅想滅念，唯一而已。滅心者，隨念隨忘，神識運行不受束縛，歸心於寂靜，直達道場。

**濟度齋：** 指回向思念至道，仰慕玄真，願福降臨無窮，災難消除未兆。又有二種，隨力修行，一是虔心，二是捨財。虔心者，完全傾盡一心，丹誠至極，燒香禮拜，唯求於道。捨財者，購買諸香油，八珍百味，營辦供具，恭請道士，以及凡品，歸心啟告；委命至真，內心泯滅六塵，外觀齊同萬景，冥心靜慮，歸誠於道，克成勝果，永契無為。

救濟存亡，拔度災苦，隨其分力，福降不差，功德重輕，各在其時。

### 齋戒功德利益

凡此諸齋日，若帝王國主、妃后大臣、公侯宰輔、道士女官、人民兆庶，或自為人燒香行道，誦經禮謝，講說此經，奏名白簡，投告山川、五嶽四瀆、山洞靈司，我常遣諸真人、神仙玉女侍巾，玉童香官使者檢錄名字，著神仙玉曆長生簿中，令厄難消除，眾苦清淨，永享無窮。

### 清妙國教化因緣

道君說：我從前在清妙國中，說法教化天人。當時有男女八萬七千人俱來詣座，同發善心，結大弘誓，為帝主國王、后妃大臣、公侯宰輔、人民土地一切眾生，常持六齋十直、三元八節、甲子庚申諸雜齋等，及建百座高台，召請道士講說此經，十二年中勤誠不退，感徹諸天。我遣左玄真人同授靈藥，各得長生；今在善積山中，並證仙道，其威儀軌則，各如三洞法。

### 十二時辰飲食宜忌

普濟稽首，問道：不知齋法以何時為宜？

道君回答說：天地運度，晝夜相推，十二時間，氣有生死，盛衰變化，善惡吉凶，人鬼不同，罪福各別，陰陽利害，自有所宜。現在具體說明，你應該仔細受持。

**吉時（宜食）：**
- **夜半（子時）：** 是諸天人食時，世人食之，增算九十，除八十一病，地官敬仰
- **雞鳴（丑時）：** 是諸天官食時，世人食之，增算一百，除病九十，地官司迎

- **日出（卯時）：** 是諸天王食時，世人食之，增算一百四十，除病一百二十，地官送迎
- **辰時：** 是諸天帝釋食時，世人食之，增算一百八十，除病一百六十，地官侍門
- **禺中（巳時）：** 是諸天真聖食時，世人食之，增算四十，除四百四病；諸天下臨，玉女散花，金童揚煙，三界五帝、日月星辰、神仙兵馬、金剛力士一時齊到；止善輟惡，名聞上清，萬天歡悅，一切獲福。**宜以此時致齋。**

**凶時（忌食）：**
- **日中（午時）：** 是諸天魔王食時，世人食之，減算九十，生八十一病，地官察身

### 慈悲分食法

若諸男女能依我法，各知所說，凡食之時，皆須先出諸食種種於前，一則獻仙聖，二饋地祇，三與鬼神，四及眾生。起此四想，是大慈悲，免諸凶惡，鬼氣不侵。散施鳥獸，遍及蟲魚，悉令飽滿，獲福無量，生死受恩。

### 受食威儀次第

普濟又問：受食有何次第？

道君回答說：先須盥漱端身，燒香行道，懺悔禮拜，發願回向。陳啟事竟，然後受食，閉目存想，身中諸神與三一同食。食之時，不得他人輒動，手有穢觸。安徐啖嚼，與氣俱進。皆當奉捧清淨巾器，勿教凡穢言笑喧聒，是名法食，始曰清齋。能依是言，諸天降駕，萬聖齊臨，眾真稱善，玉女散花，生死蒙恩，得福無量。

**重要詞彙：**
- **六齋十直：** 道教齋戒日期，每月六次齋戒和十次直日
- **三元八節：** 上元、中元、下元和立春、春分、立夏、夏至、立秋、秋分、立冬、冬至
- **甲子庚申：** 特定的干支日期，道教認為是重要的齋戒日
- **三會：** 道教三大集會日期
- **三一：** 道教修煉中的神明概念
- **六塵：** 佛道教中指色、聲、香、味、觸、法六種感官對象
