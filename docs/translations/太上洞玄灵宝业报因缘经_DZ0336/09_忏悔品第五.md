# 忏悔品第五

## 原文

忏悔品第五

尔时，大慈大悲普济真人闻太上道君说善恶因缘报对事巳，礼拜欢喜，白太上道君曰：众生愚痴暗昧，沉迷世网，不能自觉，得免罪中，流转生死，常处苦恼。伏愿大慈天尊愿见开悟，使来生男女归心善道，不落三涂。

道君语普济曰：若有善男子善女人能发自然善意，来向我前礼拜烧香，投诚忏悔，已身所犯一切恶业，及过往亡人亿曾万祖，见前父母、妻子、眷属，无量劫来舍身、受身、所在托生至于今生转无量身，所有罪咎，发心忏悔，即得灭除，身心清净，灭罪无量。若当来众生及见在男女，无量劫来举心、运心生不信敬，诽谤三宝，毁坏三宝，轻慢三宝，秽污三宝，戏弄三宝，骂詈三宝，咒诅三宝，欺惑三宝，傲𬣻三宝，唝嗃三宝；或訾毁经文，评论真伪，妄言得失，强辩是非，分别深浅，议其长短；形容不净，衣服不净，手脚不净，屦履不净，口气不净；饮酒食肉，噉嚼荤辛，口出秽言，心不恭敬，或不洗手，或不烧香，或不漱口，或不沐浴，即入玄坛灵观，便临法座，或诣道场听法闻经，触犯尊像，秽辱真圣、神仙、灵人、玉女、玉童、香官使者、金刚神王、天丁力士、护法威神；或对天尊、大道、经法、出家法身、大德真人前不生恭敬，形容傲诞，不肯礼拜、稽首归诚；或卧听法师、大德真人讲说三洞大乘经典；或酒肉荤秽，乘其车马直入道场，逼近天尊？看读经典；或见持斋受戒，永不信从，称为虚矫；或见讲说教化大众，便即破乱，谓为妖妄；或搅乱百姓，或入道场高声大语，恼触法徒；或发言举意，先论三宝，道说是非，较量真伪；或依倚三宝，方便求财；或假法教，诳惑百姓；或费用三宝财物，自养已身；或恐挟出家人，求其长短，以取财物养育妻儿；或擎经像，市中教化；或于行路聚集之间，假言功德，专将入已；或将经像假赁货卖，取财惟用供身养活妻子；或得酒肉，偿遗卖像，货赁卖经，依傍三宝求其利润；或高座说法，托以大乘，所得资财，惟供已用；或不参经戒符箓法文，便作人师，纳其财施；或广引弟子，不择贤良，授度经符，泄其真秘；或不按科律承禀师尊，自任已心，增减经戒；或尊容暴露，埃尘损坏；或屋宇倾颓，经法秽污；或破坏幡灯功德，不敬圣容，泄露经典；或盗窃宝经，同诸俗教；或私将抄写，不肯护持；或混杂凡书，曾不恭敬；或安床上枕席之间，或著屋中，一处眠寝；或同室内，臭秽狼籍，手捉酒肉语及真经；或淫欲之闻，言论三宝；或含荤秽，看读仙经；或叛道背师，自违盟誓；或转传真秘，泄漏圣文；或贵重财物，妄乱愚俗：凡诸举意一切运心，皆犯明科，违其戒律。此之罪目，无量无边。前身、今身未曾发露，爰及九玄、七祖，过去见存父母、妻子、因缘眷属，能起一念向我首陈，皆得灭除，内外清净。

道君曰：若有善男子、善女人，从无量劫来至于今日，舍身受身、父母妻子，过去见在三业六根犯诸罪目，或行苦酷，不念慈悲，杀害众生，啖食骨肉；或为父母，及以已身、兄弟、妻儿，延诸亲族，聚集乡闾，伤害众生，日无头数；或杀牛羊、驴马、猪狗、鸡豚依人养牧刍豢之属，或杀獐鹿、狐兔、虎豹、豺狼依泊山林野𤝴之属，或杀鹅鸦、乌鹊、鸠鸽、鸱鸢、凫骛、䴔䴖飞禽之属，或杀虫蛇、虺蝎、蜂虿、蚰蜒、蜈蚣、蚣蝑、蝼蚁之属，或杀鼋鼍、鱼鳖、螺蚌、虾蟆、蟹螯、蟛蜞水虫之属，或杀蜣螂、蠛蠓、𮔅蟠、螟蛉、蝎蠋、蚯蚓小虫之属，或杀蚊虻、蚤虱、蜉蝣、蛘蜎、𧍒蝡、蛸蛲化生之属，或飞鹰走狗，杀害众生；或张罗布网，杀害众生；或放火烧山，杀害众生；或穷林竭泽，杀害众生；或持弓矢，杀害众生；或持刀杖，杀害众生；或手揉脚踏，杀害众生；或坐按卧捺，杀害众生；或耕田种作，杀害众生；或打墙立壁，杀害众生；或竖屋造舍，杀害众生；或穿坑出土，杀害众生；或拆篱作障，杀害众生；或薙草平地，杀害众生；或取樵伐木，杀害众生；或摘华挽草，杀害众生；或养蚕暴茧，杀害众生；或燃灯举烛，杀害众生；或引水入穴，杀害众生；或煎汤泼地，杀害众生；或沐发浴身，杀害众生；或饮水食饭，杀害众生；或治打米麦，杀害众生；或剖胎破卵，杀害众生，或搥蝇拂蚤，杀害众生；或开口闭目，杀害众生；或举手顿足，杀害众生；或对天尊，杀害众生；或对经法，杀害众生；或对法师，杀害众生；或在观中，杀害众生；或在法座，杀害众生；或因瞋怒，杀害众生；或因愁恼，杀害众生；或因欢乐，杀害众生；或因歌舞，杀害众生；或祷祀神鬼，杀害众生；或合和汤药，杀害众生；或命绝手中，身亡刀俎；或断其头足，或剥去皮毛，或猛火烧燎，或沸汤𬊈煮；或生分子母；断其胎育；或故杀命根，令其磨灭；或杀贤人道士，百姓男女；或杀债主冤家，四方良善；或恐杀、怖杀，劫夺杀、药杀；或咒杀；骂杀，喜杀、怒杀；或自杀，或教他杀，或故杀、误杀；或口杀、手杀，意杀、想杀；或目前看杀，或传言遣杀：如此杀生，举手动身，残害无数，一发念心归诚忏悔，皆为除灭。

道君曰：若善男子、善女人从无量劫来无量劫去，见在妻子、眷属、己身、父母、九祖七玄、因缘骨肉，舍身、生身、运身，动意常行偷盗，或盗天尊财物，或盗经法财物、或盗法师财物，或盗出家财物；或盗天尊供养之物，或盗真经供养之物，或盗法师供养之物，或盗出家供养之物，或盗道场供养之物，或盗讲说供养之物，或盗玄坛供养之物，或盗灵观供养之物，或盗斋会供养之物，或盗持斋供养之物；或盗尊像修造之物，或盗修造经法之物，或盗供养法师之物，或盗修造玄坛之物，或盗修造观宇之物，或盗供养斋供清净之物，或盗护戒信心之物；或盗三宝十方之物，或盗四辈三宝之物，或盗天尊宝光上物，或盗天尊宝座上物，或盗天尊宝帐上物，或盗天尊宝盖上物，或盗天尊宝舆上物，或盗天尊宝网上物，或盗天尊宝幡上物，或盗天尊宝幢上物，或盗天尊宝冠上物，或盗天尊宝镜上物，或盗天尊宝树上物，或盗天尊宝华上物，或盗天尊宝殿上物，或盗天尊宝马上物，或盗天尊法帔上物，或盗天尊法衣上物，或盗真经宝座上物，或盗真经宝藏上物，或盗真经宝函上物，或盗真经宝蕴上物，或盗上座高尊等物，或盗妙行真人等物，或盗神仙灵人等物，或盗仙童玉女等物，或盗金刚神王等物，或盗师子辟邪等物，或盗燃灯油火等物，或盗烧香清净等物，或盗常住大众等物，或盗国家仓库等物，或盗百姓他人等物，或盗神庙鬼神等物；或于天尊前行盗，或于经法前行盗，或于灵观中行盗；或避父母行盗，或避男女行盗，或避国王行盗；或以方便取，或以迫挟取，或以势力取，或以官事取，或以欺忽取，或以诳𬤙取，或以非义取，或以强抑取，或以下脱取，或以矫妄取，或以戏弄取，或以贪爱取，或以借贷取，或以换易取，或以借贷不还取；或抵拒不偿，或罔冒欺负，或黮黯凌惑：凡非己物，丝毫毛发，华果、草木、菜叶等，起一念心，皆为偷盗。乃至路上拾得，野间逢遇，风吹鸟衔，鬼送神与；衣服卧具、器玩车马、布帛米麦，微尘分缕，不论多少，无量劫来隐藏在身，未曾发露，今能发出一切罪业，向我御前，对十方诸天尊前，哀愍摄受。今日在会大众并忏悔，弟子某家九祖以来所作众罪，未曾发露，今日今时对三宝前发心忏悔，皆得除灭。

道君曰：若有善男子、善女人，自从无量劫来，父母兄弟、妻子眷属，过去见在、舍身受身、所在生身，常行淫欲，或对天尊前起诸欲心，或对经法前起诸欲心，或对法师前起诸欲心，或见男女起诸欲心，或见衣服起诸欲心，或见华果起诸欲心，或见山林起诸欲心，或见宫殿起诸欲心，或见鸟兽起诸欲心，或见牛马鱼鳖起诸欲心；或于天尊前行诸淫欲；或于经法前行诸淫欲，或于法师前行诸淫欲，或于玄坛中行诸淫欲，或于灵观中行诸淫欲，或于道场中行诸淫欲，或于法师座下行诸淫欲，或于清净地行诸淫欲，或于私房中行诸淫欲，或于神庙中行诸淫欲，或诱人妻妾，捉人残害，离人妻子，夺人男女；或别人恩爱，隔人怜念；或六亲通同，或骨肉合会，或交奴婢，或接鸟兽，或不择妍丑，或不避贤愚，或强抑男女，或分别夫妻，或染污出家，或倮嬴露体，对三光无礼，或侵凌四辈；或男色女色乃至畜生含识之物，种种淫欲，意想梦交，无所不至，放纵六情，无量无边，结在身心，未曾忏悔。今日道场大众，各发至诚，于我御前发露忏悔，即得高真垂悯，众圣哀怜。汝等九祖以来无量罪业，五体投地，如泰山崩，拜诸上圣，万罪消灭。

道君曰：若有善男子、善女人，自从无量劫来至于今日，过去、见在父母兄弟、已身眷属，舍生受生、无量生身，常行悭贪，或于天尊大道前生诸悭贪，或于大乘经典前生诸悭贪，或于法师前生诸悭贪，或于大德真人前生诸悭贪，或于出家法身处生诸悭贪，或于斋戒供养所生诸悭贪，或于大缘事烧香供养处生诸悭贪，或于修造观宇处生诸悭贪；或生不将一文来，死后不将一钱去，一朝身死，万事皆空，几许愚痴，悭贪悋惜，今日发露，向我三宝御前同业大众冥心忏悔，前后同族一切眷属所作罪障，悉皆除灭。

道君曰：若有善男子、善女人，自从无量劫来至于今日，过去，见在父母兄弟、己身眷属，舍生、受生、无量生身，常行瞋怒，或于天尊前生诸瞋怒；或于经法前生诸瞋怒，或于出家师德真人前生诸瞋怒，或于帝王国主前生诸瞋怒，或于父母前生诸瞋怒；或见天尊十方高真诸圣尊像生诸瞋怒，或见大乘经法生诸瞋怒；或见大会法师讲说最上乘法生瞋怒心，或见金箓道场高功众官祝延圣寿生瞋怒心，或见法师讲说灵宝炼度大法生瞋怒心，或见大缘斋供广大供养生瞋怒心；或见大会持戒、设玉箓大斋、烧无量宝香供养十方尘沙贤圣，普济六道种种含识一切有情，生瞋怒心，或见乞儿贫穷、老病孤独等类生瞋怒心，或见囚徒、行旅生瞋怒心，或见牛羊、飞禽、走兽生瞋怒心；或怒天地，或怒三光，或怒国君，或怒父母，或怒鬼神，或怒奴婢，或怒畜生；若有犯、若无犯，若亲若疏，若人非人，若男若女，举心措意则生瞋怒，今日归诚，向我御前求哀忏悔，所有罪障，并得除灭。

道君曰：若有善男子、善女人，从无量劫来，或至功德福田前，生诸悭贪，或于帝王国主处生诸悭贪，或于父母男女前生诸悭贪，或于贤人长者前生诸悭贪；或见财物生悭贪心，或见饮食生悭贪心，或见田宅生悭贪心，或见车马生悭贪心，或见屋宇生悭贪心，或见奴婢生悭贪心，或见山林生悭贪心，或见华果生悭贪心，或见宫殿生悭贪心，或见城郭生悭贪心，或见贫穷生悭贪心；或见老病不肯救治，生悭贪心；或见因徒，不生救拔，生悭贪心；或见行旅饥渴困病，不济饮食、汤药，生悭贪心；乃至自巳之物、他物、众物、官司等物，毫毛丝发，草叶、华果、米麦、抄撮，悭贪悋惜不肯布施；或见天尊不肯布施，或见经法不肯布施，或见法师不肯布施，或见出家不肯布施，或见斋供不肯布施，或见持斋不肯布施，或见燃灯不肯布施，或见烧香受戒不肯布施，或见讲经说法不肯布施，或见广大道场不肯布施，或见玄坛霄殿不肯布施，或见灵观华宫不肯布施，或见造像不肯布施，或见写经不肯布施，或见广大供养不肯布施，或见诵经行道不肯布施；或见广造福田以为常住，不肯布施；或见九华宫殿、钟磬楼廊损坏，不肯布施；或见修造殿宇不肯布施，或见乞化不肯布施，或见囚徒罪人不肯布施。积财如山，心常不足，一日之活，为千年计。生时至于今日，过去、见存父母妻子、已身眷属，舍身、受身，常行愚痴。或于天尊大道处仃诸愚痴，或于大乘经典处行诸愚痴，或于大德法师处行诸愚痴，或于出家法身处行诸愚痴，或于父母君王处行诸愚痴，或于妻妾男女处行诸愚痴，或于鬼神灵圣处行诸愚痴，或于虚空贤圣处行诸愚痴；或骂詈天地，咒诅鬼神；或叛逆国君，不行仁孝；或反凶作祸，立法乱常，内淫骨肉，外杀他人；不信罪福因缘、宿命报对，谓已自然，长存不灭，不分善恶，不识贤愚，凡如此类，名愚痴也。能自觉悟，投诚忏悔，皆得除灭，常清净矣。

道君曰：若有善男子、善女人，从无量劫来至于今日，过去、见存父母妻子、己身眷属，受身、生身、无量等身，运心造意，行诸绮言，或于三洞经典行诸绮言，或于法师大德行诸绮言，或于帝主国王行诸绮言，或于父毋尊亲行诸绮言，或于贤人长者行诸绮言，或于下贱愚痴行诸绮言，或于盲聋六疾行诸绮言，或于尊贵行诸绮言，或于富豪行诸绮言；或恃长老行诸绮言，或恃豪贵行诸绮言，或恃聪明智慧行诸绮言，或恃有所识解行诸绮言；或于诸天众圣、十方神仙、星辰日月、一切威灵，凡厥举心皆行绮言，华词不实，能自觉悟，生悔谢心，来诣我前忏悔之者，皆得灭除。

道君曰：若有善男子、善女人，及父母女子、兄弟眷属，从无量劫来，舍身、生身，运心造意，绮言妄语，恶口两舌，或对天尊大、道、真经妙法、师尊父母、帝王国主、贤人道士、诸天星垣、虚空圣众一切境界，若人非人，生诸绮言妄语，恶口两舌，嫉妒胜已，斗乱是非，离人骨肉，反虚成实，转伪为真，诳误一切，莫知皈向大道高真，无量罪根发露忏悔，我悉知之，皆令清净，得入信根。

道君曰：若有善男子、善女人，舍身、生身，无量劫来，眼、耳、鼻、舌、身、意构造诸罪，杀生偷盗，邪淫放荡，悭贪瞋怒，愚痴颠倒，妄言绮语，恶口两舌，嫉妒胜巳，饮酒食肉，违犯经教，毁坏尊像，不持斋戒，不敬师尊，背真向伪，信邪倒见，祷祀鬼神，不奉三宝，訾毁唝嗃，轻慢放诞，躭声𣨼色，染习香味，沉迷触法，内造外作，展转相生，六根互起，更递成愆，罪根增长，念念成非，有无量重罪，数等尘沙，结习来久，总在身中，先缘今缘，从劫到劫，不可纪识，一发念心，皆得清净。犹如明镜，前尘、后尘、无量数尘，乃至脂垢、诸杂不净，积聚生翳，内外不明，一一遇其良匠，拂拭磨莹，表里光彻，映照远近，洞朗明净，还复如初。何以故？性本清净故。众生亦尔，神本清净，无诸滓秽，但受形之后，六贼六尘、六欲六识分别妄作，弊秽生身，外翳内兹，失其真性，流浪生死，动入罪由，报对相缘，便入地狱，受诸苦恼，万劫沉沦。吾以慈悲，怜念众生，从劫到劫，轮转人间，或夏或夷，或老或少，或凡或圣，或女或男，或贵或贱，种种化成，方便教化，度脱众生；经教流传，开悟一切，使离苦恼，永出泥犁。但众生智劣，不能解悟，我今为汝重更分别，广示因缘，令一切众生悉得解脱。

道君曰：若有善男子、善女人，常能受持此经，烧香忏悔，灭除身中一切苦恼，悉得清净，能除无量众生宿根恶业。诸天上圣、妙行真人、神仙灵人等，常欲令人烧香忏悔，则善神保护，众圣扶持。若罪结在身，内外结缚，则恶神可害，疾病缠身，水火刀兵，毒虫猛兽，枷锁牢狱，众害加身。凡夫惟知外饰面目、衣服手足，只以世水外洗浑身，而不知以法水内洗身心罪根恶业，可不愚迷。若是面目手足衣服等事，勤加洗濯，必得道真。
