# 九星都咒

## 原文

九星都咒

九曜顺行，元始徘徊。

华精莹明，元灵散开。

流盻无穷，降我光辉。

上投朱景，解滞豁怀。

得驭飞霞，腾身紫微。

人间万事，令我先知。
