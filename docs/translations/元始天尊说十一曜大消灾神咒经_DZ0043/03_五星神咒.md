# 五星神咒

## 原文

五星神咒

五星列照，焕明五方。

水星却灾，木德致昌。

荧惑消祸，太白辟兵。

镇星四据，家国利亨。

名刊玉简，字录帝房。

乘飙扇景，飞腾太空。

出入冥无，游宴十方。

五云覆盖，招神摄风。

役使万灵，上卫仙翁。
