# 🤖 AI翻譯指導規範

道教經典翻譯系統 v2.0 - AI翻譯助手專用指導文檔

## 📋 翻譯總則

### 🎯 翻譯目標
- **主要目標**: 將古代道教經典翻譯成現代中文，保持原意的同時提高可讀性
- **目標讀者**: 對道教文化有興趣的現代讀者，包括學者、修行者和一般大眾
- **翻譯風格**: 典雅而不失現代感，準確而不失流暢

### 🏛️ 基本原則
1. **忠實原文**: 準確傳達原文意思，不可任意增減內容
2. **文白對照**: 保持古文韻味的同時，確保現代讀者能理解
3. **術語統一**: 道教專業術語保持一致性
4. **文化尊重**: 尊重道教文化傳統，避免現代化過度詮釋

## 📖 翻譯規範

### 1. 📝 文本結構規範

#### 標準格式
```markdown
# [經典名稱] - 第X章

## 原文
[保持原文格式，包括段落分隔]

## 翻譯
[現代中文翻譯，保持段落對應]

## 註解
**重要詞彙：**
- [術語1]: [解釋]
- [術語2]: [解釋]

**文化背景：**
- [背景說明1]
- [背景說明2]

**翻譯要點：**
- [翻譯難點說明]
- [特殊處理說明]
```

#### 段落處理
- 原文每個自然段落對應翻譯中的一個段落
- 保持原文的邏輯結構和層次關係
- 長句可適當分割，但不改變原意

#### 一般詞彙處理原則
- **古代官職**: 翻譯為現代理解的職位，如「太守」→「郡守」
- **古代地名**: 保持原名，可在註解中說明現今位置
- **古代度量**: 翻譯為現代單位，如「一丈」→「約3.3公尺」
- **古代時間**: 翻譯為現代時間概念，如「子時」→「半夜11點到1點」

### 3. 🎨 文體風格規範

#### 語言風格
- **用詞**: 選用典雅而不艱澀的現代中文
- **句式**: 以現代中文句式為主，適當保留古文韻味
- **語氣**: 保持原文的莊重、神聖或親切語氣
- **節奏**: 注意句子長短搭配，保持閱讀節奏

#### 範例對照

**原文**: 太上曰：道可道，非常道；名可名，非常名。
**不佳翻譯**: 太上說：可以說出來的道，就不是永恆的道；可以叫出來的名，就不是永恆的名。
**標準翻譯**: 太上老君說：能夠用言語表達的道，就不是永恆不變的道；能夠用名稱指稱的名，就不是永恆不變的名。

**原文**: 夫欲修真之士，當先正其心，誠其意。
**不佳翻譯**: 想要修真的人，應該先端正心態，誠懇意念。
**標準翻譯**: 凡是想要修道的人，應當首先端正自己的內心，使意念真誠純正。

### 4. 📚 特殊內容處理

#### 咒語和真言
- **處理方式**: 保持原文，不翻譯
- **格式**: 使用特殊標記，如「咒曰：『...』」
- **註解**: 在註解中說明咒語的作用和意義

#### 詩歌和韻文
- **處理方式**: 盡量保持韻律感
- **格式**: 保持原文的分行和結構
- **翻譯**: 可適當調整語序以保持韻味

#### 引用經典
- **處理方式**: 保持原引用，在註解中說明出處
- **格式**: 使用引號標示引用內容
- **註解**: 說明引用經典的背景和意義

### 常見錯誤避免

#### ❌ 避免的翻譯錯誤
1. **過度現代化**: 不要用過於現代的詞彙替代古典概念
2. **意譯過度**: 不要為了流暢而改變原意
3. **術語不統一**: 同一術語在不同地方要保持一致
4. **文化誤解**: 不要用其他宗教或文化的概念解釋道教內容

#### ❌ 避免的表達方式
- 「老子說」→ 應用「太上老君說」或「道德天尊說」
- 「迷信」→ 應用「傳統信仰」或「宗教實踐」
- 「封建」→ 避免使用帶有政治色彩的詞彙
- 「科學的」→ 避免用現代科學標準評判古代文獻

## 📝 翻譯實踐指南

### 翻譯流程
1. **通讀原文**: 理解全文大意和結構
2. **查證術語**: 確認專業術語的準確含義
3. **初步翻譯**: 完成第一版翻譯
4. **對照檢查**: 與原文逐句對照檢查
5. **潤色修改**: 提高語言流暢性
6. **添加註解**: 補充必要的解釋說明

### 疑難處理
- **遇到不確定的術語**: 保持原文，在註解中說明不確定性
- **遇到殘缺文字**: 用「[...]」標示，在註解中說明
- **遇到異體字**: 統一為現代標準字體
- **遇到通假字**: 翻譯為本字含義，在註解中說明

### 協作規範
- **版本控制**: 每次修改都要記錄修改原因
- **討論機制**: 對有爭議的翻譯要進行討論
- **專家諮詢**: 重要術語可諮詢道教專家
- **讀者反饋**: 重視讀者的意見和建議
